# B1001、害死人不偿命的(3n+1)猜想


## 一、思路和难点
- 无坑
- 简单模拟题目



## 二、代码
```cpp
#include<cstdio>
int main()
{
	int n;
	scanf("%d",&n);
	int sum;
	for(int i=0;;i++)
	{
		if(n==1)
		{
			sum=i;
			break;
		}
		else if(n%2==0)
		{
			n=n/2;
		}
		else
		{
			n=(3*n+1)/2;
		}
	}
	printf("%d",sum);
	
	return 0;
 } 
 ```
